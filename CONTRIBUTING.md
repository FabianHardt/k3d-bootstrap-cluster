# Contributing

Contributions are welcome! Here's how to get started:

1. Fork the repository and create a branch from `main`
2. Make your changes and test them locally with `bash create-sample.sh`
3. For documentation changes, preview with `cd docs && npm ci && npm run docs:dev`
4. Open a pull request — the PR template will guide you through the rest

Please keep pull requests focused on a single topic. If you're unsure whether a change is a good fit, open an issue first.
